# SQLMate Project

SQLMate is intended to become a full SQLP preparation platform, not a simple quiz list. The product goal is to let a learner study concepts, solve exam-style questions, practice SQL tuning scenarios, review wrong answers, keep notes, and receive AI-supported explanations in one account-backed service.

## Product Pillars

- Deep SQLP concept notes with selectable highlights and personal annotations.
- Independent question banks for 1과목, 2과목, 3과목.
- SQLP practice scenarios centered on Oracle plans, SQL Trace/TKPROF, hints, indexes, joins, and rewrite.
- Wrong-answer notebook with spaced review states.
- User-owned block notes.
- Admin content workflow for draft, generated, review_required, approved, rejected, archived.
- Clear distinction between PostgreSQL execution, Oracle simulation, and future Oracle sandbox execution.
